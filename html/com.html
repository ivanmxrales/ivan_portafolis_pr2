Com s'ha fet aquest web

Aquest lloc web ha estat creat com a projecte personal per a mostrar les meves habilitats com a desenvolupador web. He utilitzat diverses tecnologies per aconseguir els objectius següents:

  • HTML5 per estructurar les pàgines web i organitzar la informació de manera semàntica.
  • CSS3 per dissenyar l'aspecte visual i assegurar la compatibilitat amb diversos dispositius.
  • JavaScript per afegir interactivitat i funcionalitat dinàmica a la web.
  • PHP per crear una estructura de backend senzilla i gestionar l'enviament de formularis.

El procés de creació d'aquesta web ha inclòs les següents etapes:

  1. Planificació i disseny de l'estructura general del lloc.
  2. Creació de les pàgines i la seva estructura amb HTML.
  3. Aplicació del disseny visual amb CSS i l'ús de flexbox per a un disseny responsive.
  4. Afegir interactivitat amb JavaScript (com la visualització de l'hora actual al capçal).
  5. Creació d'un sistema de contactes amb un formulari simple.

En resum, aquest web s'ha creat amb l'objectiu de mostrar les meves capacitats i conèixer millor les tecnologies implicades en el desenvolupament de llocs web moderns.
